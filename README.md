**MANGO**는 대학 입시를 준비하는 미술 학생들을 위한 과정 기록 및 회고 도우미 앱입니다. 

완성작뿐만 아니라 **그리는 과정** 자체를 단계별로 기록하고, 가벼운 SNS 형식으로 피드백을 주고받으며 성장할 수 있도록 돕습니다.

### 🎯 타겟 사용자
#### 팀원 Cherry의 동생 Mango는 국민대학교 시각디자인과 입학을 목표로 주말에 하루 12시간 이상 그림에 투자하는 수험생입니다.
- 🎨 입시 미술을 준비하는 수험생
- 📸 본인의 작업 과정을 체계적으로 기록하고 싶은 사람


---

## 💭 개발 배경

### 😰 문제점
- 스스로 잘하고 있는지에 대한 **불안감**
- 실력 향상을 이루어냈음에도 스스로 **성장 과정**을 확인하기 어려움

### 💡 해결 방안
- **과정 중심** 기록 시스템
- **단계별** 작업 과정 정리
- **주제별** 작품 모아보기
- **피드백** 기능을 통한 소통

---

## 주요 기능

### 1. 📝 과정 기록
- **주제별 프로젝트 생성**: 작품의 주제와 목표 설정
- **단계별 기록**: 아이디에이션, 스케치, 명암, 채색 등 각 단계를 구분하여 기록
- **상세 메모**: 각 단계에서의 고민과 시도를 텍스트로 기록

### 2. 📸 작업물 촬영 및 사진 편집
- **즉시 촬영**: 앱 내에서 바로 사진 촬영
- **5:4 비율 크롭**: 입시 미술에 최적화된 비율로 자동 편집
- **갤러리 연동**: 기존 사진도 쉽게 업로드 가능

### 3. 🗂️ 작품 아카이빙
- **다양한 보기 모드**: 프로젝트 리스트, 상세 리스트, 좋아요 그리드
- **정렬 옵션**: 최신순/과거순 정렬
- **좋아요**: 자신이 좋아하는 작품에 대한 모아보기
- **프로젝트별 분류**: 주제별로 작품 모아보기

### 4. 💬 셀프 피드백
- **댓글 기능**: 각 단계별 셀프 피드백 작성

### 5. 📱 위젯 (개발 예정)
- **홈 화면 위젯**: 바로 사진 촬영 가능
- **잠금 화면 위젯**: 빠른 접근성 제공

### 6. 🎨 개인화 설정 (개발 예정)
- **컬러 테마**: 사용자 취향에 맞는 색상 선택

### 7. 공유 기능
- **프로젝트 공유**: 자신의 프로젝트가 공유되어 가족들이 쉽게 접근하여 현재 진행상황을 확인할 수 있도록 함 (방법은 추후 추가 예정)
